# Diario de sesiones
[[masgesth]]
Fecha de revision: 2026-06-03

## Objetivo

Mantener un diario breve y acumulativo de sesiones dentro del repo para futura
integracion con Obsidian como centro de conocimiento operativo.

## Regla de uso

- una nota por dia de trabajo relevante;
- formato corto, orientado a reentrada rapida;
- no sustituye la documentacion viva principal;
- no sustituye tampoco, si se usa, al diario externo de trabajo.

## Estructura recomendada

- `YYYY-MM-DD.md` para la sesion del dia;
- `_plantilla_sesion.md` como base reutilizable.

## Contenido minimo recomendado

- foco del dia;
- cambios cerrados;
- validaciones hechas;
- riesgos o pendientes;
- primer siguiente paso sugerido.
