#### 🔌 API Integration

Conexión entre aplicaciones mediante APIs (REST, GraphQL, SOAP). Permite que una app lea/escriba datos en otra automáticamente. Base de cualquier automatización.

**Ejemplo:** Make conecta con API de HubSpot (CRM), extrae contactos nuevos, y los envía a API de Mailchimp para email marketing. 2 APIs comunicándose.
